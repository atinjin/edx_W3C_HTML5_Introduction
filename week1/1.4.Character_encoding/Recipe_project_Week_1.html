How to generate Github pages

This is the way to generate Github pages. Follow these steps.

  1. Step. SignUp/SignIn Github.com
  2. Step. Create the repository
  3. Step. Change the setting to "Github page".
  4. Steo. Check-out the repository to local
  5. Step. Edit the index.html
  6. Step. Commit the index.html
  7. Step. Push the changes
  8. Step. Repeat
